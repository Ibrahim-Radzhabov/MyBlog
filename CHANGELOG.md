# Changelog

Все заметные изменения по релизам проекта.

## v0.1.2 - 2026-03-12

### Added
- Добавлена аналитика поиска каталога через `search_events`.
- Добавлены виджеты в админке:
  - топ поисковых запросов за 30 дней;
  - запросы без результатов за 30 дней.
- Добавлена миграция `supabase/migrations/003_search_events.sql`.

### Changed
- Обновлены и стабилизированы e2e-тесты для production-прогона.
- README и релизный чеклист переведены и актуализированы на русском.

## v0.1.1 - 2026-03-11

### Fixed
- Нормализация `NEXT_PUBLIC_SITE_URL` и `ADMIN_EMAIL` из env.

## v0.1.0 - 2026-03-09

### Added
- Первый production-safe MVP Prompt Atlas:
  - публичные страницы каталога;
  - защищенная админка;
  - CRUD для промптов/категорий/тегов;
  - аудит действий администратора;
  - базовые e2e-проверки.
